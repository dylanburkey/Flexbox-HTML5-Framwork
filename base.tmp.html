dylanburkey.com will be back soon with ‹basis›

Basis uses CSS Flexbox to structure the layouts included in the framework. The focus of BASIS is developing a semantic HTML5 and CSS/CSS3 Framework - utilizing Flexbox for all template structures.

BASIS is currently in alpha staging with portions of the framework still being brainstormed. If you are interested in BASIS and would like to learn more - visit the BASIS page on a GitHub.

The following features are standard in BASIS

  • a. Basis is developed in HTML(5), CSS(3), Javascript.
  • b. Basis comes standard with rich HTML5 semantic templates.
  • c. Basis uses PHP to create custom Flexbox templates. ** THIS IS ONLY IF YOU PLAN TO INSTALL BASIS AS A COMPLETE FRAMEWORK AND TEMPLATING APP. **
  • d. Basis is Mobile/Tablet/TV/ and Clock Radio Ready.
  • e. Basis supports IE 6 and above, Opera, Firefox, Chrome, Safari, Chromium, Canary and Aruara.
